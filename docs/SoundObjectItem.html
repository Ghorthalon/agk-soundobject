Class: SoundObjectItem

SoundObjectItem(file, Callback, tag)

One item in the soundObject array of sounds. If you're not modifying this library you need not worry about this constructor as the sound will be returned when you call the soundObject.create() method.

Constructor

new SoundObjectItem(file, Callback, tag)

Parameters:
Name Type Description
file string the file to be loaded.
Callback function on loading complete.
tag number Tags of 1 won't be cleared when you flush the soundObject cache, 0 will be cleared.
